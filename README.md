## Getting Started

### Prerequisites

-   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/gobwah/bank_modern_app.git
    ```
2. Install dependencies
    ```sh
    npm install
    ```
